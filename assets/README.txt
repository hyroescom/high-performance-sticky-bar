WordPress.org Plugin Repository Assets

This directory contains assets that will be visible on your plugin's WordPress.org page.

REQUIRED ASSETS:

1. banner-772x250.jpg/png - Main plugin banner image (displayed at the top of your plugin page)
   - Dimensions: 772x250 pixels
   - File format: JPG or PNG

2. icon-256x256.jpg/png - Plugin icon (displayed in search results and as favicon)
   - Dimensions: 256x256 pixels
   - File format: JPG or PNG

OPTIONAL BUT RECOMMENDED:

3. banner-1544x500.jpg/png - Retina version of the banner
   - Dimensions: 1544x500 pixels
   - File format: JPG or PNG

4. icon-128x128.jpg/png - Medium size icon
   - Dimensions: 128x128 pixels
   - File format: JPG or PNG

SCREENSHOT IMAGES:
Place your screenshot images in the main /assets directory:
- screenshot-1.png - The settings page
- screenshot-2.png - The sticky bar in action
- screenshot-3.png - The color picker

These images should match the descriptions in your readme.txt file.
